# Learn Table – Juego educativo de tablas de multiplicar

Learn Table es un juego educativo interactivo diseñado para que los niños aprendan las tablas de multiplicar de manera divertida. El juego combina práctica con niveles progresivos, colores llamativos y un sistema que desbloquea nuevas tablas a medida que se completan correctamente las anteriores.

## Características:

- Selección de tablas y niveles progresivos.
- Feedback inmediato al responder correctamente.
- Diseño colorido y adaptado para niños.
- Registro del progreso usando localStorage.

## Tecnologías utilizadas

- React + TypeScript
- Vite
- Shadcn UI (componentes UI)
